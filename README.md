Calm Companion

Calm Companion is a web-based emotional wellbeing support platform designed for individuals with neurodevelopmental conditions such as ADHD and Autism.
The system uses facial emotion recognition to detect emotional states like stress, distraction, or calmness and provides supportive activities to help users regulate emotions and improve focus.

It also includes a Parental Control System that allows caregivers to monitor emotional patterns, guide users, and receive alerts when intense emotions persist.
